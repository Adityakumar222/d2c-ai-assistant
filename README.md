# D2C AI Growth Assistant

AI-powered backend service that helps D2C (Direct-to-Consumer) merchants improve pricing, marketing strategies, and profitability using LLMs.

## 🚀 Features
- FastAPI backend
- OpenAI API integration
- Generates actionable business insights
- Structured output (Pricing, Marketing, Profit)

## 🧠 Use Case
Designed to simulate real-world decision support for e-commerce merchants by analyzing inputs like pricing, ad spend, and conversions.

## ⚙️ Tech Stack
- Python
- FastAPI
- OpenAI API
- Docker (optional)

pip install -r requirements.txt
